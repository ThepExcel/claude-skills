## Color Formula

| Element | Color | Priority |
|---------|-------|----------|
| **Outfit** | Vivid Orange | MAIN — ทุกรูปต้องมี |
| **Accent 1** | Blue | เล็กๆ — ต่างหู, สร้อย, เล็บ, แหวน |
| **Accent 2** | Red | เล็กๆ — ดอกไม้, ริบบิ้น, รองเท้า |
| **Background** | Neutral Grey | Concrete, steel, white porcelain |

**Principle:** Ti (提) saturation hierarchy — สีสดเฉพาะจุดเน้น ไม่กระจาย

**Rule:** Blue + Red accent ไม่เกิน 1-2 ชิ้นต่อภาพ

## What Works (from 19 selected)

### Camera Distance — ต้องมีทุกระยะ

| Distance | % ที่เลือก | Notes |
|----------|-----------|-------|
| Extreme Close-up | 16% | ตา, ริมฝีปาก, profile — texture + intimacy |
| Close-up | 16% | หน้า + ไหล่ — mirror, upside down |
| Medium / Upper body | 26% | กอดอก, มือจับผม, lean forward |
| Full Body | 26% | นอน bench, บันได, dutch angle |
| Wide | 16% | Corridor, bathtub, laundromat — context |

**Key:** กระจายเท่าๆ กัน = variety ของ set

### Camera Angle — Bird's Eye เป็น Favorite

| Angle | เลือก | Mood |
|-------|-------|------|
| **Bird's eye (top-down)** | 21% (favorite) | Vulnerability, geometric, editorial |
| Eye level | 32% | Standard, direct connection |
| Slight high angle | 21% | Intimate, approachable |
| Dutch angle | 5% | Edgy, dynamic, uncomfortable |
| Low angle | 5% | Power, confidence |
| Profile 90° | 5% | Sculptural, classical |
| Macro | 11% | Detail, hyperreal |

### Pose — Relaxed > Power

| Pose | เลือก | |
|------|-------|-|
| **นอน/เอน (lying/reclining)** | 32% (favorite) | Relaxed vulnerability |
| นั่ง (sitting) | 21% | Casual, grounded |
| ยืน (standing) | 16% | Confident but less common |
| Close-up only | 32% | Face/expression focus |

**Rule:** Simple pose 1-2 action > complex pose

### Expression

| Expression | เลือก |
|------------|-------|
| **สายตาตรงกล้อง + neutral/confident** | 53% (signature) |
| มองออก / ไม่มองกล้อง | 26% |
| ตาปิด / half-closed | 11% |
| ยิ้ม | 11% |

**Signature:** สายตาตรง + ไม่ยิ้ม (Newton influence)

### Anti-Boring Elements (ต้องมีอย่างน้อย 1)

| Element | ตัวอย่างที่ work |
|---------|-----------------|

## Scenes That Work

### Full Body / Wide

| Scene | Why |
|-------|-----|
| Concrete bench | Simple, clean, S-curve body line |
| Bathtub | Water texture layer, intimate |
| Laundromat | Mundane + surreal = Bourdin |
| Concrete corridor | Vanishing point, architectural |
| Stairs | Geometric shadow patterns on body |
| Parking garage | Industrial, Newton shadow play |

### Close-up

| Technique | Why |
|-----------|-----|
| Mirror reflection | Frame-in-frame, private moment |
| Upside down (lying, shoot from head) | Disorientation = attention |
| Profile with rim light | Sculptural, classical |
| Hair wind (from below chin) | Caught moment, raw |
| Extreme eye macro | Hyperreal detail, orange eyeshadow |

---

## What Doesn't Work

| Avoid | Reason |
|-------|--------|
| Complex/contorted pose | AI generates unnatural results |
| Too many colors | Dilutes orange dominance |
| Thematic costumes (matrix, etc.) | Breaks editorial cohesion |
| Extra fingers in complex hand poses | Re-seed if happens |
| Scaffolding/climbing structures | Too weird, concrete bench > scaffolding |
| Heavy makeup | Natural > editorial makeup for this face |
